# 🧱 Minno

Minno, **API + UI** olmak üzere 2 katmanlı olarak geliştirilen, modern ve sürdürülebilir mimari prensipleri hedefleyen bir projedir.

Bu proje yalnızca “çalışan bir uygulama” üretmek için değil;

✅ Clean Code  
✅ Dependency Injection  
✅ DTO (Data Transfer Object) yapıları  

gibi profesyonel yazılım geliştirme yaklaşımlarıyla tasarlanmıştır.

---

## 🚀 Proje Amacı

Gerçek dünya senaryolarına uygun,  
AI destekli,  
modern backend geliştirme pratiği sunmak.

---

## 🤖 Yapay Zekâ Entegrasyonları

Projede çeşitli AI servisleri kullanılmıştır:

✨ **OpenAI** → Girilen malzemelere göre yemek tarifi önerisi  
💬 **OpenAI** → Müşteri mesajlarına otomatik cevap üretimi  
🛡️ **Hugging Face** → Mesajlarda toksiklik tespiti  
⚡ **OpenAI + SignalR** → Gerçek zamanlı AI Chat Bot  
📊 **OpenAI** → Örnek veri setleri üretimi  
🎨 **Anthropic Claude** → Tasarımsal iyileştirme önerileri
